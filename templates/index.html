Welcome to MI District and School Demographics Warehouse

Background

MI district and school demographics warehouse is an api set up to help the public undestand the demographic breakdown of MI schools and districts, as well as related information about resources and socioeconomic status. The data is free to use and may be helpful for people looking to research the recent history of the school system, or build dashboards related to equity in education. The financial data found at this API is taken from the US Census Bureau by way of a Kaggle dataset. The numbers are from 2016. Some processing was done on the metrics to normalize financial figures to the number of students in district. The school demograpics data was scraped from a look up tool on Mlive. This data is from 2017.



Instructions for Developers

This api is quite straightforward to use. Simply put in the URL for the data you want. We do not require an api key. Most keys in our JSONs are self explanatory, but financial data is normalized to enrollment so "fed_rev_per_s" is federal revenue per student, and all financial statements are reported in thousands of dollars. Each JSON is structured as an object with a single key, which gives a list of objects. There is one object for every school or district.
